

# 忘憂君府 Minecraft 伺服器專案

本 GitHub 儲存庫包含「饕餮月音的忘憂君府」Minecraft 伺服器所使用的資料包與資源包

## 專案結構
```
yueyin-minecraft-server
├─ datapack          伺服器遊戲機制的資料包
└─ resourcepack      伺服器使用的資源包
```
